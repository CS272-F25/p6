# Stop the Cap!

Stop the Cap! is a multi-page informational website that explores the history, function, and modern cultural role of hats. The site examines how hats have evolved over time, how they differ across cultures and eras, and how they continue to serve both practical and stylistic purposes today.

The project was built as a static website using HTML, CSS, and JavaScript, with a focus on clean design, accessibility, and responsive layout.

## Purpose
The purpose of this website is to provide an accessible and engaging overview of hats as both functional objects and cultural symbols. Visitors can learn about historical trends, modern styles, and practical considerations such as sizing and everyday use.

## Technologies Used
- HTML5 (semantic markup)
- CSS3 (global stylesheet)
- Bootstrap 5 (layout, navigation, responsiveness)
- JavaScript (DOM manipulation, user interaction)
- Fetch API (external data loading)
- LocalStorage (data persistence)
- Git & GitHub
- GitHub Pages

## Key Features
- Multi-page site with consistent navigation
- Responsive layout for mobile, tablet, and desktop
- Hat size calculator with saved results
- Modern styles page using real external data via fetch
- Image gallery showcasing common hat styles
- Timeline page showing the evolution of hats over the past 1000 years
- FAQ page answering common questions about hats
